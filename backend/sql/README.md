# Backend SQL Scripts

This directory contains PostgreSQL functions and scripts for the LineUp application.

## Directory Structure

### `search/`

Search functions for the search feature.

Files:

- `search_for_you.sql` - Aggregated "For You" tab
- `search_people.sql` - User profile search
- `search_collaborations.sql` - Collaboration request search
- `search_services.sql` - Service type search
- `search_tags.sql` - Tag/metadata search
